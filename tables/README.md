# Scripts

Standalone Lua scripts and AOB (Array of Bytes) patterns for use in Cheat Engine.

## Usage

1. Open Cheat Engine.
2. Go to `Table → Show Cheat Table Lua Script`.
3. Paste or load the script.
4. Click **Execute**.
